240 FPS & 240 TPS Setup

All macros in this collection were recorded at 240 FPS and 240 TPS. To use them correctly, your game settings should be set to those values as well.

Step 1

Open the Settings button, then find the Graphics setting. Go to Advanced, check Unlock FPS if it is not already enabled, and set FPS to 240. Click Apply when done.

Step 2

Open the Silicate mod with the F or Alt key, and set TPS to 240.

How to run these macros

To load a macro with the Silicate replay mod:

  1. Choose a macro from this website.
  2. Copy the `.slc` file.
  3. Paste it into the correct replay directory for your platform:

Windows:
C:\Users\[user]\AppData\Local\GeometryDash\geode-persistent\peony.silicate\replays

Linux (Proton):
~/.steam/steam/steamapps/compatdata/322170/pfx/drive_c/users/steamuser/AppData/Local/GeometryDash/geode-persistent/peony.silicate/replays

macOS (Proton):
~/Library/Application Support/Steam/steamapps/compatdata/322170/pfx/drive_c/users/steamuser/AppData/Local/GeometryDash/geode-persistent/peony.silicate/replays

  1. Launch Geometry Dash.
  2. Press F or Alt to open the replay menu.
  3. Select the macro, then click Load and then Play.

If the replay folder doesn’t exist, confirm that Geode and Silicate are installed correctly.
