# Geometry Dash 240 FPS & TPS Macro Collection

A curated set of **240 FPS & TPS Geometry Dash macros** for practice, learning, and experimentation. Each macro is distributed as a **.slc replay file** and organized so it is easy to browse by level type, style, and difficulty.

> This collection is intended for educational and showcase purposes only, and it does not encourage cheating.

## 📁 Repository Structure

The collection is organized into three main groups:

### Main Levels
Official Geometry Dash levels, sorted by difficulty folders such as:
- Easy
- Normal
- Hard
- Harder
- Insane
- Demon

### Rated Custom Levels
Custom levels that are rated, grouped by:
- Classic
- Platformer

Each style then contains difficulty folders such as Easy, Normal, Hard, Harder, Insane, Easy Demon, Medium Demon, Hard Demon, Insane Demon, and Extreme Demon.

### Unrated Custom Levels
Custom levels that are not rated, also grouped by:
- Classic
- Platformer

Each style contains the relevant difficulty folders for that section.

Every macro is clearly named, and macros that collect coins will include that behavior where applicable.

## 🎮 Using the Macros

To load a macro with the **Silicate** replay mod:

1. Choose a macro from the website or repository.
2. Download or copy the `.slc` file.
3. Paste it into the correct replay directory:

- **Windows:**  
  `C:\Users\[user]\AppData\Local\GeometryDash\geode-persistent\peony.silicate\replays`

- **Linux (Proton):**  
  `~/.steam/steam/steamapps/compatdata/322170/pfx/drive_c/users/steamuser/AppData/Local/GeometryDash/geode-persistent/peony.silicate/replays`

- **macOS (Proton):**  
  `~/Library/Application Support/Steam/steamapps/compatdata/322170/pfx/drive_c/users/steamuser/AppData/Local/GeometryDash/geode-persistent/peony.silicate/replays`

**Note:** On Linux and macOS, Geometry Dash runs inside a Proton prefix, so the folder structure mirrors Windows.

4. Launch Geometry Dash.
5. Press **F** or **Alt** to open the replay menu.
6. Select the macro, then click **Load** and **Play**.

If the folder does not exist, double-check that **Geode** and **Silicate** are installed correctly.
